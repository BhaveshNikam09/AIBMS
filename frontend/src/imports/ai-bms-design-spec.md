Design a high-fidelity, production-ready landing page and authentication flow for an AI-powered Business Management System (AI-BMS) built for Indian MSMEs.

This is a serious fintech-grade SaaS product. The UI must feel premium, trustworthy, intelligent, and enterprise-ready. Avoid generic startup template aesthetics.

GLOBAL DESIGN SYSTEM

• 12-column responsive grid
• Light theme primary
• Background: soft neutral (#F7F9FC style)
• Primary color: deep navy / charcoal
• Accent color: subtle emerald/teal for AI highlights
• Typography: Inter or IBM Plex Sans
• Clean spacing rhythm (8px grid system)
• Border radius: 10–12px
• Soft layered shadows
• Subtle 200–300ms transitions
• No flashy gradients
• No neon or playful UI

--------------------------------------------------

LANDING PAGE STRUCTURE

Hero Section:
• Full-width section
• Left side:
  - Headline: “Your AI Financial Operating System for MSMEs”
  - Short subtext explaining value
  - Primary CTA: “Get Started”
  - Secondary CTA: “Book Demo”
• Right side:
  - High-fidelity dashboard mockup preview inside device frame
• Subtle background gradient or soft pattern
• Floating micro-animations (very subtle)

Trusted By Section:
• Logos placeholder row
• Clean horizontal layout

Features Section (Card Layout):
Display 6 core features in structured grid:
• AI Dashboard Insights
• Digital Cashbook with Voice Entry
• Multi-Branch Management
• ITR Analysis & Tax Estimation
• Document Intelligence
• AI Chatbot

Each card should include:
• Minimal icon
• Short title
• One-line description
• Soft hover elevation

How It Works Section:
• 3-step horizontal layout
  1. Connect & Add Data
  2. AI Analyzes Business
  3. Get Insights & Decisions
• Subtle connecting line animation

Business Benefits Section:
• Split layout
• Left: short text blocks
• Right: analytics UI preview
• Emphasize:
  - Save Time
  - Reduce Compliance Risk
  - Improve Cash Flow

Security & Trust Section:
• Show security badge icons
• Short explanation:
  - Secure cloud storage
  - Encrypted data
  - Role-based access

Final CTA Section:
• Strong centered message
• Button: “Start Managing Smarter”
• Clean background with subtle contrast

Footer:
• Product links
• Features
• Contact
• Privacy Policy
• Terms

Layout:
• Split screen design

• Smooth route transitions between Landing → Login → Signup
• Button hover micro-lift

OVERALL FEEL

The entire experience must feel like:

“A secure, intelligent financial system built for real business owners.”

Avoid:
• Generic bootstrap admin look
• Oversized rounded playful UI
• Bright gradient-heavy startup vibe
• Overcrowded sections

This is a premium fintech SaaS platform for Indian MSMEs.
