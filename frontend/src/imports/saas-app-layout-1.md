Design a complete enterprise-grade SaaS application layout for an AI-powered Business Management System (AI-BMS) built for Indian MSMEs.

This is NOT a generic admin template. It must look premium, structured, and fintech-level professional.

Global Layout Requirements:
- 12-column responsive grid system
- Collapsible left sidebar navigation
- Sticky top navbar
- Clean spacing hierarchy
- Clear visual depth (subtle shadows, elevation)
- Minimal, calm, trustworthy design
- No flashy gradients
- No startup-style playful UI

Sidebar:
- Vertical navigation with icons + labels
- Active state indicator (subtle glow or accent line)
- Sections:
  Dashboard
  Digital Cashbook
  Multi-Branch Management
  ITR Analysis
  Document Intelligence
  AI Chatbot
  Reports & Analytics
  Settings

Top Navbar:
- Business selector dropdown
- Branch selector
- Notification bell
- User profile avatar
- Minimal search bar
- Sticky behavior on scroll

Main Layout Structure:
- Content area with clear page header section
- Breadcrumb navigation
- Page title + subtitle
- Action button aligned right

Dashboard Page Layout:
- First row: 4 KPI cards (Revenue, Expenses, Profit, Cash Balance)
- Second row:
  Left (8 columns): Cashflow line chart
  Right (4 columns): AI Business Health insight panel
- Third row:
  Receivables summary card
  Compliance timeline card

Digital Cashbook Page:
- Top: Add Transaction button + Voice Entry button
- Filter bar below header
- Main area: Transaction table
- Right side panel: Running balance summary

Multi-Branch Page:
- Branch comparison cards at top
- Role-based access table
- Permission toggle UI

ITR Analysis Page:
- Document upload drag-and-drop area
- Extracted summary cards
- Tax estimation panel
- Deduction insights section

AI Chatbot Page:
- Left: Conversation history panel
- Right: Main chat interface
- Structured answer formatting (Summary, Explanation, Action Steps)

Reports Page:
- Report type selector cards
- Filter configuration panel
- Preview table
- Export buttons (PDF / CSV)

Design Philosophy:
- Premium fintech SaaS aesthetic
- Soft neutral background
- Deep navy or charcoal primary
- Clear typography hierarchy
- Subtle micro-interactions
- Elegant spacing rhythm
- Professional and trustworthy

The layout must feel like a serious financial operating system, not a basic dashboard template.
